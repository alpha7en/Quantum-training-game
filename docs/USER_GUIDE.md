# Руководство пользователя

## Режимы

1. **Экзамен** — работа с билетами, теорией, сборкой схемы и проверкой ответа.
2. **Задачи** — набор квантовых челленджей с ограничениями и автопроверкой.

## Конструктор схем

- Перетаскивайте гейты из панели в ячейки сетки.
- Для многокубитного управляемого гейта:
  - поставьте `C` на управляющих кубитах;
  - поставьте целевой гейт в том же столбце.
- Для параметрических гейтов:
  - тап по ячейке → кнопка «⚙️ Настр» (ярлык настройки);
  - откроется Bottom Sheet с ползунком и полем выражения.

## Начальные состояния

- Тап по бейджу `Q_i init`.
- Выберите:
  - чистое состояние: `|0⟩`, `|1⟩`, `|+⟩`, `|−⟩`, или custom (`θ, φ`);
  - смешанное состояние: вероятность `p0` (автоматически `p1 = 1 - p0`).

## Симуляция

- Нажмите **«Запустить схему»**.
- Просматривайте шаги по тактам (`t=0...N`).
- Доступные представления:
  - вектор состояния (если состояние чистое),
  - матрица плотности,
  - вероятности базисных исходов,
  - сферы Блоха по кубитам.

## Shots (статистические измерения)

- Выберите число запусков (100, 500, 1000, 8000).
- Запустите симуляцию shots.
- Получите гистограмму распределения результатов.

## Справка по гейтам

- Нажмите `ⓘ` на гейте в панели или «ⓘ Справка» в меню ячейки.
- Откроется описание с матрицей и действием на базисные состояния.

## Верификация задач

- В режиме задач нажмите **«Проверить решение»**.
- Приложение сравнит результат схемы с целевым:
  - state-задачи: fidelity по плотностной матрице;
  - unitary-задачи: fidelity унитарных матриц.
- Решение считается верным при fidelity > 0.999.
