# Quantum Training Game (MIPT)

Продвинутый веб‑тренажёр по квантовым вычислениям для магистров: конструктор схем, симуляция чистых и смешанных состояний, учебные билеты и игровой режим задач.

## Что уже реализовано

- Квантовый движок на TypeScript (без бэкенда), комплексная арифметика, унитарная эволюция, матрицы плотности.
- Конструктор схем с drag-and-drop (pointer events), сеткой по тактам, контролями и визуальными связями.
- Гейты: `H, X, Y, Z, Rx, Ry, Rz, P, C, Mz, Mx, My`.
- Параметризованные гейты через mobile-friendly Bottom Sheet + парсер выражений (`pi/2`, `-pi/4`, `2*pi/3`).
- Режим симуляции: вектор состояния (если состояние чистое), матрица плотности, вероятности, сферы Блоха.
- Режим Shots: гистограмма статистических измерений после N запусков.
- Режим задач: ограничения на гейты/параметры, фиксированные элементы, проверка через fidelity.
- PWA: `manifest.json`, `service worker`, оффлайн-кэширование.

## Технологии

- React + TypeScript + Vite
- KaTeX для формул и матриц
- PWA (Service Worker + Manifest)

## Быстрый старт

```bash
npm install
npm run dev
```

Сборка:

```bash
npm run build
```

Локальный просмотр production-сборки:

```bash
npm run preview
```

## Структура проекта

- `src/quantumMath.ts` — математика, линейная алгебра, парсеры, fidelity.
- `src/TaskManager.tsx` — движок прогона схем, задачи, shots.
- `src/CircuitBuilder.tsx` — конструктор схем, drag-and-drop, модалки состояний.
- `src/GateConfig.tsx` — Bottom Sheet для параметров гейтов.
- `src/GateHelp.tsx` — справка по гейтам и формулы.
- `public/sw.js` + `public/manifest.json` — PWA.
- `docs/` — документация по архитектуре, использованию и соответствию ТЗ.
